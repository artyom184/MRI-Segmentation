## 2. Mask R-CNN (ResNet-50 + FPN)

### Архитектура (Section 2.4 из статьи)
- **Backbone**: ResNet-50 (предобученный на ImageNet)
- **Neck**: Feature Pyramid Network (FPN)
- **Head**: классификация + bounding box + маска
- **Loss**: L = L_cls + L_box + L_mask (формула 1 из статьи)
- **Предобучение**: COCO dataset

## 3. Алгоритм анализа деформаций

Реализован в файле `deformity_analysis.py` (v2).

### Шаги алгоритма (Section 2.5):
1. **Постобработка маски** (Section 2.5.1): OPEN + CLOSE + largest connected component
2. **Поворот** (Section 2.5.1): `cv2.minAreaRect` → компенсация лордоза
3. **Разделение** маски на верхнюю/нижнюю части по центру масс (Section 2.5.2)
4. **Гистограмма** ненулевых пикселей по X + Гауссово сглаживание (Section 2.5.2)
5. **6 ключевых точек** (Section 2.5.2, Fig. 6):
   - Au, Al — передние (максимум в первых 10%)
   - Pu, Pl — задние (максимум в последних 10%)
   - Mu, Ml — центральные (минимум ±10% от центра масс)
6. **3 высоты** (Section 2.5.3, формулы 4–6): Ah, Ph, Mh — евклидово расстояние
7. **Классификация Genant** (Section 2.5.4, формулы 7–10):
   - Normal: < 20%
   - Mild (Grade 1): 20–25%
   - Moderate (Grade 2): 25–40%
   - Severe (Grade 3): > 40%
